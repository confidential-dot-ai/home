Confidential AI runs AI workloads — inference, agents, training, and more — inside
hardware-backed Trusted Execution Environments, so prompts, responses, model weights, and
credentials stay encrypted end to end, even from the infrastructure operator. These docs
cover the products you build on, the concepts underneath them, and the research behind it all.
