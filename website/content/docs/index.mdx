Everything for building on Confidential AI. **c8s** turns any Kubernetes cluster into a
confidential one — your workloads' data and artifacts stay encrypted in hardware-backed TEE
memory the whole time they're in the cluster, and that guarantee is provable to any third
party. Start with the four steps below, or jump into the concepts, whitepapers, and APIs.
